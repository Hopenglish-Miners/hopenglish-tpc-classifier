# spark-heroku-example

Code for tutorial: [https://sparktutorials.github.io/2015/08/24/spark-heroku.html](https://sparktutorials.github.io/2015/08/24/spark-heroku.html)

HOW TO RUN
1. Install maven
2. Instal ruby
3. install foreman [gem install foreman]

TO RUN LOCAL
foreman start

TO DEPLOY TO HEROKU
mcn heroku:deploy
